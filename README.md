# Crypto Estates

A modern, fully functional real estate website that allows users to browse apartment listings and complete purchases using cryptocurrency payments.

This is a Next.js application bootstrapped with `create-next-app` and configured for Firebase Studio.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by editing the homepage at `src/app/page.tsx`.
